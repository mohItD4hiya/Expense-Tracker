# 💰 Expense Tracker

A simple and responsive Expense Tracker web application built using **HTML, CSS, and JavaScript**. This project helps users manage their daily expenses by recording transactions, tracking spending, and monitoring their financial activities in an easy-to-use interface.

## 🚀 Features

- Add income and expense transactions
- Track total balance in real time
- View income and expense summaries
- Delete transactions when needed
- Responsive design for desktop and mobile devices
- Simple and intuitive user interface
- Local storage support to save data between sessions

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (
- Local Storage API

## ⚙️ How It Works

1. Enter a transaction description.
2. Add the transaction amount.
   - Positive value = Income
   - Negative value = Expense
3. Click **Add Transaction**.
4. The application automatically updates:
   - Current Balance
   - Total Income
   - Total Expenses
5. Transactions are stored locally and remain available after page refresh.
